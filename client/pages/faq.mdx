## Unter Linux und MacOS

bashscript: das bashscript im Stammverzeichnis kann zur Ausführung des Programms verwendet werden.
Java-Version über 11 sollte installiert sein und das Skript sollte im Stammverzeichnis des Projekts ausgeführt werden

### Linux

```bash
./bashscript
```

### MacOS

```bash
sh bashscript
```

### Ausführung des Java-Programms ohne script

```bash
./mvnw clean compile exec:java
```

### Test auszuführen:

```bash
./mvnw test
```

im Stammverzeichnis des Projekts ausgeführt werden

### für Java-Docs

```bash
./mvnw javadoc:javadoc
```

## Unter Windows: mvnw.cmd
